Practical 1
Aim
Install Unity Hub and execute basic C# scripts using conditions and loops. 
PRACTICAL NO 1
Steps
1.	Open Unity Hub.
2.	Click New Project → 2D Core → Create.
3.	Go to Hierarchy → Right Click → Create Empty Object.
4.	Name it ScriptObject.
5.	Go to Assets → Create → C# Script.
6.	Name it BasicScript.
7.	Open the script in Visual Studio.
8.	Write the required code (if-else, switch, loops).
9.	Save the script.
10.	Drag the script onto the ScriptObject.
11.	Click Play.
12.	View output in Console Window.

Practical 2
Aim
Implement Translation, Rotation and Scaling on a 2D object. 
PRACTICAL2ofUNITY
Steps
1.	Create 2D Project in Unity.
2.	In Hierarchy → Right Click → 2D Object → Sprite → Square.
3.	Rename it Player.
4.	Select Player → Add Component → Rigidbody2D.
5.	Set Gravity Scale = 0.
6.	Go to Assets → Create → C# Script.
7.	Name script TransformScript.
8.	Write code for translation, rotation, and scaling.
9.	Attach script to Player.
10.	Click Play to see movement.

Practical 3
Aim
Implement Jump effect and movement. 
PRACTICAL NO 3_UNITY
Steps
1.	Create 2D project.
2.	Add Circle sprite in Hierarchy.
3.	Rename it Player.
4.	Add Rigidbody2D component.
5.	Set Gravity Scale = 1.
6.	Create Square object and rename it Ground.
7.	Add BoxCollider2D to ground.
8.	Create new C# Script → JumpMove.
9.	Write code for jump and movement.
10.	Attach script to Player.
11.	Click Play.
12.	Press Arrow keys to move and Space to jump.

Practical 5
Aim
Implement Surface Effector and Area Effector. 
PRACTICAL NO 5
Steps
Surface Effector
1.	Create Circle object (Player).
2.	Add Rigidbody2D + CircleCollider2D.
3.	Create platforms using squares.
4.	Add BoxCollider2D to platforms.
5.	Add Surface Effector 2D to platform.
6.	Set Surface Arc = 180.
7.	Create JumpMove script.
8.	Attach script to player.
9.	Press Play.
10.	Player automatically moves on platform.
Area Effector
1.	Create large square.
2.	Add Area Effector 2D.
3.	Add BoxCollider2D.
4.	When player enters area → force is applied.

Practical 7
Aim
Add Audio Source in Unity. 
Practical 7_UNITY
Steps
1.	Download MP3 audio file.
2.	Drag audio file into Assets folder.
3.	Select a GameObject (camera or object).
4.	Click Add Component → Audio Source.
5.	Drag audio file into Audio Clip.
6.	Enable Play On Awake.
7.	Click Play.
8.	Audio will start automatically.

Practical 10
Aim
Implement Animation in Unity. 
PRACTICAL NO 10_unity
Steps

3.	Drag character into scene.

5.	Create Animator Controller.
6.	Add animations to controller.
7.	Create transitions between animations.
8.	Create script Animation_Script.
9.	Attach script to character.
10.	Run scene.
11.	Press W → Walk.
12.	Press D → Run.
